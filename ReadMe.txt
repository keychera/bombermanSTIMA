 THE REAL README FOR THIS PROJECT

This is bot for Bomberman Game Engine from Entelect Challenge 2016 that is also a task for a course in ITB
this is the stuff: https://github.com/EntelectChallenge/2016-Bomberman

this is the SampleBot folder that contain the solution files and folders plus the shared bot.json
to use this bot, download the whole thing, put bot.json OUT of this folder, put this folder and bot.json in a new folder, use that new folder for referencing the bot when using the game engine. for more info about using the bot with the game engine look it up on the link above.
